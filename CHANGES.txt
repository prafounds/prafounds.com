HOW TO RUN LOCALLY:
=============================================================================

BASIC SETUP:
$ cd PraFounds-Site
$ npm install
$ export DATABASE_URL="postgresql://localhost/prafounds"
$ npm run dev

EXPECTED RESULT:
- Server starts on http://localhost:5000 (or configured PORT)
- All animations load smoothly
- Floating orbs visible in background
- Cards respond to hover with lift & glow effects
- Gradient backgrounds display on all sections

CUSTOMIZATION:
- Edit colors: client/src/index.css (:root variables)
- Edit animations: @keyframes in client/src/index.css
- Adjust orbs: client/src/components/BackgroundEffects.tsx
- Edit section styling: client/src/pages/Home.tsx
